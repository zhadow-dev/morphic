# Generated Assets

Real screenshots captured from the **built** `examples/notes_workspace` app
(Flutter + the Morphic runtime), via `PrintWindow` on each surface's own HWND —
no desktop, no other windows, no synthetic mockups.

Reproduce with:
```
# from morphic/ (repo root)
powershell -NoProfile -ExecutionPolicy Bypass -File tools/capture_notes.ps1   # captures each surface
powershell -NoProfile -ExecutionPolicy Bypass -File tools/compose_hero.ps1    # composes the hero still
```

## Honest provenance

- **Generated by the agent:** all four files above. Each window is genuine pixels
  from the running app. `notes-multiwindow.png` arranges the three *real* captures
  on a dark gradient — it composites real windows, it does not fake them.
- **Empty states are real, not staged.** The Editor/Inspector show "Select a note"
  / "No note selected" because no note is selected at boot.

## Still requires a human on a Windows machine

(See `morphic-site/asset-plan/` for full specs.)

- **A "selected-note" hero** (Editor showing a note body, Inspector showing a live
  word count) — needs a real click in the List. Driving a believable interactive
  click is intentionally **not** done here (risk of clicking other apps on a live
  desktop; it's the one capability reported as NO).
- **All motion:** every GIF and video, the cold-start/spawn clips, the launch
  trailer, and the spatial shoot. No video encoder is available (no ffmpeg) and
  interactions can't be scripted convincingly.
